# useLifecycles

生命周期调用函数

```jsx
const App = () => {
  useLifecycles({
    create: () => {
      console.log('CREATE')
    },
    mount: () => {
      console.log('MOUNTED')
    }, 
    unmount: () => {
      console.log('UNMOUNTED')
    } 
  });

  return <div></div>;
};
```
